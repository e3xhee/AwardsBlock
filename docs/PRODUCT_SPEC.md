# AwardBlock Product Spec

## Overview

AwardBlock은 해커톤, 공모전, 빌더 프로그램의 공식 수상 결과를 검증 가능한 기록으로 남기고, 수상자가 ERC-20 상금을 직접 Claim할 수 있게 만드는 Web3 수상 아카이브 플랫폼이다.

사용자 화면의 Award Block은 온체인 수상 기록, 트랜잭션, 이벤트 로그, ERC-20 Claim 내역, 오프체인 메타데이터, 메타데이터 해시 검증 결과를 하나로 묶어 보여주는 서비스 단위다. 수상 기록 하나가 블록체인의 개별 블록이 된다는 의미로 설명하지 않는다.

## Users

### Organizer

운영자는 지갑으로 로그인하고 행사, 프로젝트, 수상 결과, 수상자별 배정 금액, 초대 링크, 상금 예치, 온체인 확정, 정정 기록을 관리한다.

### Recipient

수상자는 초대 링크로 접속해 지갑을 연결하고 본인의 배정액을 확인한 뒤 ERC-20 상금을 직접 Claim한다.

### Public Viewer

일반 조회자는 로그인 없이 행사, 프로젝트, Award Block, 지갑별 수상 이력, 트랜잭션 링크, 메타데이터 검증 결과를 확인한다.

## Core Flow

1. 운영자가 지갑 서명으로 로그인한다.
2. 운영자가 행사를 생성한다.
3. 운영자가 수상 프로젝트를 등록한다.
4. 운영자가 수상 결과와 팀원별 배정 금액을 작성한다.
5. 시스템이 수상자별 초대 링크를 생성한다.
6. 수상자가 초대 링크에서 지갑을 등록한다.
7. 운영자가 ERC-20 상금을 컨트랙트에 예치한다.
8. 운영자가 수상 결과를 온체인에 확정한다.
9. 수상자가 배정된 상금을 Claim한다.
10. 공개 Award Block 페이지에서 수상 정보와 검증 상태를 조회한다.

## MVP Scope

### Included

- 하나의 EVM 네트워크
- ERC-20 한 종류를 선택한 수상별 지급
- 지갑 연결과 서명 인증
- 행사, 프로젝트, 수상 결과 생성
- 팀원별 배정 금액 입력
- 초대 링크 생성과 1회 사용 처리
- 수상자 지갑 등록
- 상금 예치와 온체인 확정
- 수상자별 Claim
- 공개 행사, 프로젝트, Award Block, 지갑 프로필 페이지
- 메타데이터 해시 검증
- 정정 기록 연결
- 로컬 테스트용 MockUSDC

### Excluded

- NFT 또는 Soulbound Token 발급
- 다중 체인과 크로스체인 지급
- 법정화폐 결제, KYC, 세금 계산
- 이메일 자동 발송과 소셜 로그인
- 심사위원 투표와 프로젝트 제출 시스템
- Merkle Claim, IPFS 업로드, 가스비 대납
- 모바일 앱

## Product Principles

- 검증 가능성과 정확성을 기능 수보다 우선한다.
- 온체인 데이터와 오프체인 메타데이터의 책임을 분리한다.
- 블록체인 기록만으로 수상 내용이 사실이라고 설명하지 않는다.
- 어떤 운영자 지갑이 기록했는지 항상 표시한다.
- 금액은 최소 단위 정수 문자열과 bigint로 처리한다.
- 실제 자산 사용 전에는 별도 보안 감사를 전제로 한다.
